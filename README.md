# YE-D1
A Simple desktop clock using SSD1306 and D1MINI

------------


### Requirements
* D1 mini
![D1Mini](https://imgur.com/iGFpiMS.jpg)

* SSD1306 128*64 0.96'
![SSD1306](https://imgur.com/8hlHmZm.jpg)

* A 3D Printer
files in Cura/

![2](https://imgur.com/kwzZsgz.jpg)

### Library
* Adafruit_GFX
* Adafruit_SSD1306


### Connection table
| D1 Mini  | SSD1306 |
| :--------: |:-------:|
| 5V       | VDD     |
| GND      | GND     |
| 5 SCL    | SCK     |
| 4 SDA    | SDA     |
